Каталог для размещения основных раширений системы - компонентов.
Каждый компонент состояит из файлов и каталогов.

Компонент:
- компонент.php - основной файл действий компонента.
- компонент.админ.php - основной файл действий панели управления данным компонентом.
- компонент.инфо.php - файл сборной информации о компоненте. Информация о авторе, версии.
- компонент.md5.php - файл md5 хешей файлов входящих в состав компонента. Требуется для проверки целосности и версионности.
- компонент.парамс.xml - файл параметров компоненте XML формате. Содержит расширенные параметры конфигурации компонента.
- компонент.меню.php - файл генерации доступных типов меню компонента. С его помощью можно создавать разные типы меню на разные фукнции компонента.
- компонент.роуты.php - файл конфигурации роутов компонента.
- компонент.виджет.php - виджеты для вставки в шаблон.
- /languages/ - каталог содержащий языковые файлы компонента.
- /view/ - каталог содержащий шаблоны отображения компонента, лицевую часть.
- /adminview/ - каталог содержащий шаблоны отображегия компонента, часть панели управления
- /media/ - медиа-файлы компонента
- /media/css/ - файлы стилей компонента
- /media/js/ - ява-скрипт файлы компонента
- /media/swf/ - флэш-файлы компонента
- /docs/ - каталог содержащий дикументацию и справку по компоненту
